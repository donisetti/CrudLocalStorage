LocalStorage + JQuery + Bootstrap

Neste exemplo foi construido um CRUD (create, read, update e delete) utilizando o LocalStorage como repositório de dados.

Foi utilizado o Bootstrap para o layout, o Jquery Validation para validar o formulário, o Jquery Datatables para construir a grid de resultados com opções de ordenação, busca e filtro, usei ainda um plugin para lançar as mensagens ao usuário o Jquery Growl
